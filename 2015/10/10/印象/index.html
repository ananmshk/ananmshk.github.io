印象

file *fp = fopen(“文件路径”，“文件操作方式”)
文件操作方式：r 读
r+ 读和写
w 写
w+ 读和写 如果文件不存在则创建一个
a :pen for writing. The file is created if it does not exist 文件指针指向结尾
a+

文件关闭:
fclose(fp) 先将缓冲区中尚未存盘的数据写盘(fflush)，然后撤销存放该文件信息的结构体，最后令指向该文件的指针为空

文件读写操作
字符读写函数：fgetc :从文件指针stream所指的文件中读取一个字符，并返回该字符
fputc:将给定字符c写入到文件指针stream所指向的文件中:
fgetc( FILE stream ) 返回值为int，参数为文件指针.若果指针指向文件末尾或者读取错误，则返回EOF。
fputc( int c,FILE stream);参数：int c被写入的字符，FILE *stream写入文件的指针。返回值：正确写入返回被写入的字符，如果错误返回EOF.

按行读写函数：fgets 和fputs
fgets(char restrict str,int size,FILE restrict stream)(用来存储所得数据的地址，存储数据的大小，将要读取的文件指针)
返回值：
fputs(要写入的字符指针，要写入的文件指针)；返回值为一个非负数整数，错误则返回EOF

数据块读写函数：fread和fwrite
fread(用于存放将要读取的数据地址，每次读取的数据大小，读取多少次，被读取的文件指针)
fread(void restrict ptr, size_t size, size_t nitems，FILE restrict stream);
返回值：实际读取次数
fwrite(用于写入的数据的地址，每次写入的大小，写入次数，要写入的文件指针)

格式化读写函数：fscanf和fprintf
